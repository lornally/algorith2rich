### 系鞋带

- 两种
  - 平结/蝴蝶结/爱结/赫拉克勒斯结(比较好看的系法)
  - 外行平结/祖母结(一般的鞋带系法)
- 两个结组合, 
  - 每次都有左右两个方向
  - 因此有4个组合结果
  - 可以导致形成: 平结/祖母结
